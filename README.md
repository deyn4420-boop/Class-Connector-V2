# ClassConnect — Desktop App (EXE Version)

## Build .EXE in One Click

Step 1 — Install Python from https://python.org (tick "Add Python to PATH")
Step 2 — Open the classroom_desktop folder in terminal
Step 3 — Double-click BUILD.bat

That's it. Both .exe files are built automatically.

## Output after building:
dist/ClassConnect_Teacher/ClassConnect_Teacher.exe  ← Teacher runs this
dist/ClassConnect_Student/ClassConnect_Student.exe  ← Students run this

IMPORTANT: Share the entire folder, not just the .exe

## Troubleshooting:
- "Windows protected your PC" → Click More info → Run anyway
- Students can't connect → must be on same Wi-Fi network
- Antivirus blocks → add dist folder to exclusions
